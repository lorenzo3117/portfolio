Projects

Personal project: EchoDiary

EchoDiary is a fully featured blog application that allows users to share their thoughts and experiences.

Users can register an account and verify their email address. After that, they can create, edit, and delete posts with a rich text editor. It is also possible to follow other users and get email and in-app notifications when a new post is published.

The admin user can appoint moderators that have access to a moderation dashboard and the ability to review posts and ban bad-behaving users.

Commenting and liking posts is the next feature I'm working on.

The web application is built with Laravel, daisyUI (Tailwind CSS) and PostgreSQL.

Flemish government: Verkeersmodellen

Traffic models are a tool used to study the effects of infrastructural, spatial, or policy measures on traffic.

The Traffic Models team at the Department of Mobility and Public Works uses different types of models, mainly strategic models for areas where traffic demand is still unknown, and dynamic models for more detailed analysis of traffic flow, congestion, and safety. These models support Flemish policy by enabling studies and scenario analyses, especially in collaboration with government entities.

The web application is built with Java (Spring Boot), Angular and PostgreSQL. AWS is used for hosting the application.

Flemish government: DCBaaS

DCBaaS is a platform that allows organizations to independently request and manage X.509 digital certificates for their applications or APIs.

The platform supports full self-service, enabling authorized users to create, renew, revoke, and monitor certificates without centralized intervention. It integrates with the Flemish government's identity management system (IDM) to control user access rights and provides notifications about certificate expirations.

The web application is built with Java (Spring Boot), React and PostgreSQL. AWS is used for hosting the application and Jenkins for CI/CD.

DCBaaS

Flemish government: Chiron

Chiron is the central ride database system for taxi service operators in Flanders.

As a taxi service owner, you are required to send information about your rides to Chiron, which acts as a central database for collecting and managing ride data.

The web application is built with Java (Spring Boot), Angular and PostgreSQL. AWS is used for hosting the application and for CI/CD.
